# Character Type Counter

A Python script that counts letters, digits, spaces, and special characters in text.

## Description
This project analyzes user input and categorizes each character into different types.

## Features
- Counts letters
- Counts digits
- Counts spaces
- Counts special characters
- Simple and beginner-friendly

## Technologies Used
- Python

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:

   python character_counter.py

4. Enter text when prompted

## Example

Input:
Hello 123!

Output:
Letters: 5
Digits: 3
Spaces: 1
Special Characters: 1
